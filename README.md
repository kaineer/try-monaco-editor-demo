# TLDR

## Как поставить

 * clone
 * npm install
 * browse ./index.html

## Что показывается

 * Вставка кода
 * Смена позиции курсора
 * Выбор кода
 * Удаление выбранного кода
 * Блокирование изменения содержимого редактора пользователем
 * Перехват фокуса (в рамках браузера) в процессе вывода кода
